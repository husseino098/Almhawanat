# 🐄 سوق الحيوانات – تطبيق ويب لبيع وشراء الحيوانات في العراق

تطبيق ويب متكامل (SPA) مكتوب بـ **HTML و CSS و JavaScript النقي**، يهدف إلى تسهيل عملية بيع وشراء الحيوانات بين المستخدمين في العراق.  
جميع البيانات تُخزَّن محلياً في متصفحك باستخدام `localStorage`، مما يعني عدم الحاجة إلى خادم أو قاعدة بيانات خارجية لتجربة التطبيق.

---

## ✨ المميزات الرئيسية

- **إدارة الإعلانات**: إضافة إعلانات جديدة مع رفع صور الحيوانات، وعرضها في الصفحة الرئيسية.
- **فلتر وبحث متقدم**:  
  - بحث نصي (بالاسم، المحافظة، أو العنوان).  
  - تصفية حسب **المحافظة** (جميع المحافظات العراقية الـ 18).  
  - تصفية حسب **صنف الحيوان** (أكثر من 20 صنفاً).
- **المفضلة**: حفظ الإعلانات التي تعجبك بنقرة زر القلب (تظهر في صفحة التفاصيل).
- **حسابات المستخدمين**:  
  - إنشاء حساب جديد.  
  - تسجيل الدخول والخروج.  
  - عرض إحصائيات شخصية (عدد إعلاناتي، عدد المفضلات).
- **إدارة إعلاناتي**:  
  - عرض إعلاناتك المنشورة فقط.  
  - حذف أي إعلان بسهولة مع عرض عدد المعجبين به.
- **واجهة عربية بالكامل (RTL)**: تصميم عصري ومتجاوب مع جميع أحجام الشاشات (خاصة الهواتف).

---

## 🚀 طريقة التشغيل والتثبيت

1. **أنشئ ملف `index.html`** في مجلد على جهازك.
2. **انسخ كود التطبيق** (الموجود في الرد السابق) والصقه داخل الملف.
3. **افتح الملف** باستخدام أي متصفح حديث (مثل Chrome، Firefox، أو Edge).
4. **ابدأ الاستخدام**!  
   - المستخدم الافتراضي (أحمد) جاهز لتجربة التطبيق فوراً.  
   - يمكنك تسجيل الخروج من الإعدادات وإنشاء حساب جديد.

> 📌 **ملاحظة**: التطبيق لا يحتاج إلى اتصال بالإنترنت بعد تحميل الصفحة، باستثناء تحميل مكتبة الأيقونات (Font Awesome) من CDN.

---

## 📱 واجهات التطبيق (الصفحات)

| الصفحة | الوصف |
| :--- | :--- |
| **الرئيسية** | عرض جميع الإعلانات مع شريط فلتر وبحث. البطاقات تحتوي على: صورة، اسم الحيوان، السعر، المحافظة، وعلامة "توصيل" إن وجدت. |
| **إعلاناتي** | عرض إعلانات المستخدم الحالي فقط، مع إمكانية الحذف وعرض عدد المعجبين. |
| **المفضلة** | عرض الإعلانات التي تم حفظها (بدون أزرار إضافية للحفاظ على التصميم النظيف). |
| **الإعدادات** | إدارة الحساب (تسجيل الدخول، إنشاء حساب، تسجيل الخروج) وعرض الإحصائيات. |
| **إضافة إعلان** | نموذج متكامل يشمل: صورة، صنف الحيوان، العدد، الوزن، السعر (دينار عراقي)، رقم الهاتف، المحافظة، عنوان تفصيلي، وخيار التوصيل. |
| **تفاصيل الإعلان** | عرض جميع بيانات الإعلان مع أزرار: الإضافة للمفضلة، الاتصال، والحذف (للمالك فقط). |
